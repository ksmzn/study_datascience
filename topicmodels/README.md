# TopicModel
(Python|R) implementations of LDA(latent Dirichlet Allocation):

- LDA (Collapsed Variational Bayes)
- (WIP)HDP-LDA (Collapsed Variational Bayes)
- (not yet)NDP-LDA

# [Python]Requirements:

- Python(>=3.5)
- numpy(>=1.10.4)
- scipy(>=0.17.0)
